# MOF-Bandgap-Prediction-using-Machine-Learning
This project focuses on predicting the electronic bandgap of Metal–Organic Frameworks (MOFs) using machine learning models trained on a large-scale dataset of approximately 20,000 structures. The goal is to explore how structural and compositional features influence material properties and to compare the performance of different ML approaches.
The reference data: https://figshare.com/articles/dataset/QMOF_Database/13147324/12
